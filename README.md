# 🦖 Dino Run - Unity Clone

A simple clone of the classic Chrome Dino Game built with Unity.

## 🎮 Gameplay

- Control a dinosaur running through an endless desert
- Jump over cacti and dodge obstacles
- Score points by surviving as long as possible
- Speed increases over time for added difficulty

## 🕹️ Controls

| Action | Key |
|--------|-----|
| Jump | `Space` or `Up Arrow` |

## ✨ Features

- Endless runner gameplay
- Parallax scrolling background
- Randomly generated obstacles
- Progressive difficulty (increasing speed)
- Score tracking with high score
- Game over and restart functionality
- Simple pixel-art style graphics
- Day/night cycle (optional)

## 🛠️ Built With

- **Unity** (version: 2021.3 LTS or newer)
- **C#** for game logic
